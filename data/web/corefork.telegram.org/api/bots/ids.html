Bot API dialog IDs

The id field of peers Â» (i.e. users, chats, channels) contain three distinct sequences of unique 64-bit IDs used to identify a specific user, chat or channel.

The ID sequences of users, chats and channels overlap, so it is a good idea to transform the peer IDs to bot API dialog IDs as specified below.

A bot API dialog ID is a single, unique peer ID sequence derived from the user, chat and channel ID sequences, maintaining uniqueness across all of them.

It's a good idea to transform peer IDs to bot dialog API IDs even if you do decide to use separate databases to store info about peers, as it will make IDs more visually recognizable both for you and your users, as well as guarantee compatibility with the bot API.

Example implementation: tdlib (bot API), MadelineProto.

User IDs

User IDs in the MTProto API range from 0 to 0xffffffffff (inclusive).

To convert MTProto user IDs to bot API dialog IDs, do the following:

$botApiId = $userId; // Do nothing

To convert bot API dialog IDs to MTProto user IDs, do the following:

assert($botApiId > 0 && $botApiId <= `0xffffffffff`);
$userId = $botApiId;

Chat IDs

Chat IDs in the MTProto API range from 0 to 999999999999 (inclusive).

To convert MTProto chat IDs to bot API dialog IDs, do the following:

$botApiId = -$chatId;

To convert bot API dialog IDs to MTProto chat IDs, do the following:

assert($botApiId < 0 && $botApiId >= -999999999999);
$userId = -$botApiId;

Supergroup/channel IDs

Supergroup/channel IDs in the MTProto API range from 0 to 997852516352 (inclusive).

To convert MTProto chat IDs to bot API dialog IDs, do the following:

$botApiId = -1000000000000 - $chatId;

To convert bot API dialog IDs to MTProto chat IDs, do the following:

assert(-1000000000000 <= $botApiId < 0 && $botApiId !== );
$userId = -$botApiId;
